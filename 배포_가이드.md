# 📱 핸드폰에서 접속할 수 있게 배포하기 (Streamlit Cloud)

목표: `내앱이름.streamlit.app` 같은 주소로 어디서든 폰으로 접속. PC 꺼져 있어도 OK.

준비물: 이메일 주소 하나. 그게 전부입니다.

---

## 1단계 — GitHub 계정 만들기 (5분)

1. 브라우저에서 **https://github.com** 접속
2. 우측 상단 **Sign up** 클릭
3. 이메일, 비밀번호, 원하는 username 입력 (예: `hilar-trader`)
4. 이메일 인증 링크 클릭 → 끝

> 팁: username은 URL에 노출됩니다. 너무 개인적인 걸 피하세요.

---

## 2단계 — GitHub에 코드 올리기 (10분)

### 2-1. 새 저장소(Repository) 만들기

1. GitHub 로그인 후 우측 상단 **+ 버튼 → New repository**
2. 설정:
   - Repository name: `tqqq-signal` (원하는 이름)
   - Description: `TQQQ 자동 매매 신호 앱`
   - **Private** 선택 ⚠️ (Public 하면 전 세계에 공개됨)
   - `Add a README file` **체크 해제** (이미 있으니까)
3. **Create repository** 클릭

### 2-2. 내 컴퓨터 코드 올리기 — 가장 쉬운 방법

GitHub 사이트에서 **직접 업로드**할 수 있습니다.

1. 방금 만든 저장소 페이지에서 **Add file → Upload files** 클릭
2. 폴더 창을 열고 이 파일들만 선택해서 드래그:
   - `app.py`
   - `backtest.py`
   - `data.py`
   - `metrics.py`
   - `charts.py`
   - `presets.py`
   - `storage.py`
   - `signal_engine.py` ← 새로 만든 파일
   - `requirements.txt`
   - `strategy.md`
   - `TQQQ_사용자_매뉴얼.html`
   - `README.md`
3. **절대 올리지 말 것:**
   - `__pycache__/` 폴더 (자동 생성 파일)
   - `*.bak_hedge` 파일들 (백업)
   - `saves/` 폴더 (개인 저장 결과)
   - `my_position.json` (있다면 ⚠️ — 개인 포지션 정보)
4. 화면 아래 "Commit changes" 버튼 클릭

> 💡 `my_position.json`을 Public이나 남이 볼 수 있는 곳에 올리면 안 됩니다. Private repo라도 신중하게.

### 2-3. `.gitignore` 파일 추가 (안전장치)

1. 저장소 페이지에서 **Add file → Create new file**
2. 파일명: `.gitignore`
3. 내용 붙여넣기:
   ```
   __pycache__/
   *.pyc
   *.bak_hedge
   saves/
   my_position.json
   .streamlit/secrets.toml
   ```
4. Commit new file 클릭

---

## 3단계 — Streamlit Cloud에 배포 (5분)

1. **https://share.streamlit.io** 접속
2. **Continue with GitHub** 클릭 → GitHub 계정 연결 (권한 허용)
3. **New app** 버튼 클릭
4. 설정:
   - Repository: 방금 만든 `username/tqqq-signal` 선택
   - Branch: `main`
   - Main file path: `app.py`
   - App URL: 원하는 주소 (예: `tqqq-hilar` → `tqqq-hilar.streamlit.app`)
5. **Deploy** 클릭
6. 2~3분 기다리면 앱이 실행됩니다.

---

## 4단계 — 핸드폰에서 접속 + 홈화면 아이콘 만들기

### iPhone (Safari)
1. 배포된 URL 접속 (예: `tqqq-hilar.streamlit.app`)
2. 하단 **공유 버튼** 탭 → **홈 화면에 추가**
3. 이름 정하고 추가 → 끝. 이제 아이콘 누르면 앱처럼 실행됩니다.

### Android (Chrome)
1. 배포된 URL 접속
2. 우측 상단 **⋮ 메뉴** → **홈 화면에 추가** (또는 **앱 설치**)
3. 아이콘 이름 정하고 추가 → 끝.

---

## 5단계 — 포지션 입력 & 저장

1. 폰/PC 어디서든 배포 URL 접속
2. **📱 오늘의 신호** 탭 → 내 포지션 입력 → **저장** 버튼
3. 포지션은 `my_position.json`에 저장됩니다.

### ⚠️ Streamlit Cloud의 저장 한계

Streamlit Cloud 무료 플랜은 앱이 재시작되면 **앱이 만든 파일이 지워집니다.** 그래서:

**해결책 A (가장 간단) — JSON 백업 복사해두기**
- 포지션 저장 후 화면의 "🔐 포지션 백업 / 복원 (JSON)" 섹션 펼치기
- JSON 텍스트를 폰 메모앱이나 노션 같은 곳에 복사해 두기
- 앱이 초기화되면 → 텍스트 붙여넣고 "복원하기" 버튼

**해결책 B — GitHub에 커밋 (복잡)**
- 포지션 바꿀 때마다 GitHub 저장소에 `my_position.json` 수정해서 커밋
- Private repo라면 상대적으로 안전

초반엔 **A**로 하시고 나중에 불편하면 B로 업그레이드하세요.

## 6단계 — 매일 사용법

1. 아침에 (또는 미국장 마감 후 새벽) 폰에서 홈화면 아이콘 탭
2. 앱이 자동으로 Yahoo Finance에서 최신 데이터 받아옴 (10~30초)
3. **📱 오늘의 신호** 탭에서:
   - 오늘의 행동 (매수/매도/관망)
   - 매매할 주수 / 금액
   - 다음 주요 이벤트 예상가
4. 신호대로 증권사 앱에서 주문

## 문제 해결

**Q. 배포 후 에러가 뜬다**
- Streamlit Cloud 화면 아래쪽 로그를 확인. 보통 `requirements.txt`에 패키지 누락.

**Q. 데이터가 안 받아와진다**
- Yahoo Finance가 간헐적으로 느릴 수 있음. 새로고침.

**Q. 포지션이 자꾸 사라진다**
- 해결책 A(JSON 백업)가 근본 대책입니다. 바꾼 직후 JSON을 복사해 두세요.

**Q. 전략 파라미터를 폰에서도 바꿀 수 있나요?**
- 네, 사이드바에서 조정 가능. 단 모바일에선 사이드바가 햄버거 메뉴(≡) 안에 있습니다.

**Q. Public repo에 올려도 되나요?**
- 코드는 괜찮지만 `my_position.json`(개인 투자 정보)은 반드시 `.gitignore`로 제외하세요.
