# TQQQ 백테스트 프로그램

QQQ 신호 기반 TQQQ 매매 전략을 40년간 백테스트하는 인터랙티브 웹 앱입니다.

---

## 처음 실행하는 방법 (단계별)

### 1단계 — 명령 프롬프트(CMD) 열기

`Windows키 + R` 누르고 `cmd` 입력 후 엔터

### 2단계 — 프로젝트 폴더로 이동

```
cd "D:\backtest for TQQQ"
```

### 3단계 — 패키지 설치 (처음 한 번만)

```
pip install -r requirements.txt
```

설치에 1~2분 걸릴 수 있습니다. 오류 없이 완료되면 다음 단계로 넘어가세요.

### 4단계 — 앱 실행

```
streamlit run app.py
```

잠시 후 브라우저가 자동으로 열리며 앱이 표시됩니다.
(자동으로 안 열리면 브라우저에서 `http://localhost:8501` 주소 입력)

### 5단계 — 앱 종료

CMD 창에서 `Ctrl + C` 누르면 종료됩니다.

---

## 앱 사용법

- **왼쪽 사이드바** 슬라이더로 전략 파라미터 조정 → 차트가 즉시 업데이트
- **첫 실행 시** Yahoo Finance에서 데이터를 다운로드하므로 인터넷 연결 필요 (30초~1분)
- 이후 실행에서는 캐시된 데이터 사용 (빠름)

---

## 파일 구조

```
D:/backtest for TQQQ/
├── app.py          ← Streamlit 앱 (여기서 실행)
├── data.py         ← 데이터 다운로드 및 합성
├── backtest.py     ← 전략 엔진
├── metrics.py      ← 성과 지표 계산
├── charts.py       ← 차트 생성
└── requirements.txt
```

## 자주 묻는 질문

**Q: 앱이 안 열려요**
A: Python이 설치되어 있는지 확인하세요. CMD에서 `python --version` 입력 시 버전 숫자가 나와야 합니다.

**Q: 데이터 다운로드가 느려요**
A: Yahoo Finance 서버 상태에 따라 다릅니다. 잠시 기다려 주세요.

**Q: 슬라이더를 바꿔도 반응이 없어요**
A: 브라우저를 새로고침 하거나 CMD에서 앱을 재시작해 보세요.
